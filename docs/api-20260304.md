## 数据模型变更

### 1. vm/models.go - Group 结构增强
```go
type Group struct {
    Name        string    `json:"name"`
    Description string    `json:"description"`
    VMs         []string  `json:"vms"`
    Owner       string    `json:"owner"`        // 创建者用户名
    IsShared    bool      `json:"is_shared"`    // 是否共享
    SharedWith  []string  `json:"shared_with"`  // 共享用户列表
    CreatedAt   time.Time `json:"created_at"`
    UpdatedAt   time.Time `json:"updated_at"`
}
```

## 关键功能说明

### 批量执行命令
- 支持并行/串行执行模式
- 限制并发数为10
- 支持超时控制
- 自动记录命令历史

### 资源监控
- 通过 SSH 执行系统命令获取指标
- CPU使用率、内存使用率、磁盘使用率
- 系统负载、运行时间、网络流量
- 批量资源获取支持状态评估（ok/warning/error）

### 分组共享
- 创建组时自动记录 owner
- 支持设置 is_shared 和 shared_with
- 查询时根据用户权限过滤
- 管理员可查看所有组

### 命令模板
- 支持变量替换（${name}格式）
- 变量类型：string, number, select
- 支持共享模板
- 权限控制：只有创建者和管理员可修改

### 命令历史
- 自动记录批量执行命令
- 支持分页查询
- 可按VM名称和命令内容搜索
- 记录执行时长和输出

### 跨服务器文件传输
- 通过 SSH 从源服务器下载文件
- 上传到目标服务器
- 自动创建目标目录
- 支持大文件传输
